3 run_MC_CW_IRSL_LOC Monte-Carlo Simulation for CW-IRSL (localized transitions) Runs a Monte-Carlo (MC) simulation of continuous wave infrared stimulated luminesence (CW-IRSL) using the generalized one trap (GOT) model. Localized transitions refer to transitions which do not involve the conduction or valence band. These transitions take place between the ground state and an excited state of the trapped charge, and also involve an excited state of the recombination center. 0.1.0 Sebastian Kreutzer, IRAMAT-CRP2A, UMR 5060, CNRS - Université Bordeaux Montaigne (France)
